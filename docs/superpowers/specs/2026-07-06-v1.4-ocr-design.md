# Sill v1.4 — OCR 텍스트 추출 설계

날짜: 2026-07-06
상태: 승인됨

## 목표

썸네일 우클릭 → 이미지에서 텍스트를 인식(OCR)해 화면 중앙 카드에 표시.
ESC로 취소, Enter/⌘C로 클립보드 복사 후 닫기.

## 범위

포함:
- 우클릭 메뉴에 "텍스트 추출" 추가
- Vision(`VNRecognizeTextRequest`)으로 한국어+영어 인식, 의존성 추가 없음
- 결과를 중앙 오버레이 패널에 표시, 키보드로 복사/취소

제외:
- 인식 결과 편집, 영역 선택 재인식, 언어 수동 선택 (YAGNI)

## 제약

- **최소 타깃 macOS 14** (한국어 인식은 Vision revision 3 = Sonoma+). Package.swift/Info.plist 상향.
- 외부 패키지 의존성 금지. 자동 테스트 없음 — 게이트는 `swift build` 성공 + 수동 GUI 검증.
- 기존 우클릭 항목(복사/저장) 동작 불변.

## 컴포넌트

### TextRecognizer (신규, UI 없음)
`static func recognize(_ image: NSImage, completion: @escaping (String) -> Void)`
- NSImage→CGImage, 백그라운드 큐에서 `VNRecognizeTextRequest`(`.accurate`, `usesLanguageCorrection`, `recognitionLanguages = ["ko-KR","en-US"]`).
- 관찰 결과의 top candidate를 줄바꿈으로 연결, 메인에서 completion. 실패/빈 결과는 빈 문자열.

### OCROverlayController (신규, 결과 표시 전용)
- `canBecomeKey`를 override한 borderless NSPanel + SwiftUI 카드(`OCRResultView`, 텍스트 스크롤·선택 가능, 480×360).
- `show(text:)`: 앱 활성화 → 마우스가 있는 스크린 중앙 배치 → makeKeyAndOrderFront → 로컬 keyDown 모니터 설치.
- 키: ESC(53)=닫기, Return(36)/Enter(76)=복사+닫기, ⌘C=복사+닫기.
- 닫기: orderOut + 모니터 해제. 빈 텍스트면 "(인식된 텍스트 없음)" 표시.

### 배선
- 우클릭 메뉴 "텍스트 추출" → 클로저를 `AppDelegate → StackPanelController → StackView → ThumbnailView → DragArea → DragNSView`로 전달(기존 copy/saveAs와 동일).
- AppDelegate가 `OCROverlayController` 소유. 클로저: `TextRecognizer.recognize(shot.image) { overlay.show(text: $0) }`.

## 데이터 흐름

우클릭 → `onExtractText(shot)` → 백그라운드 OCR → 메인 콜백 → 오버레이 표시 → 키 입력으로 복사/취소.

## 검증

- `swift build` 성공.
- 한글/영문 섞인 스크린샷에서 텍스트 인식, 중앙 표시, Enter·⌘C 복사, ESC 취소 수동 확인.
