# Sill v1.1 — 안정화와 설정 (설계)

날짜: 2026-07-05 / 상태: 승인됨

## 범위

1. **설정 창 (SwiftUI)** — 메뉴바 메뉴 "설정..." (⌘,)로 열림. 항목:
   - 로그인 시 자동 실행: `SMAppService.mainApp` register/unregister. 실패 시 토글 원복 + 로그.
   - 스택 위치: 왼쪽/오른쪽 라디오. UserDefaults 저장, 변경 즉시 패널 반영.
2. **주 모니터 고정** — 패널 위치 기준을 `NSScreen.main`(포커스 화면) → `NSScreen.screens.first`(주 디스플레이)로.
3. **핫키 등록 실패 알림** — `HotkeyManager.register`가 OSStatus 반환, 실패 시 실행 시점에 NSAlert 1회.

## 제외 (YAGNI)

마진 수치 설정(16pt 고정), 스택 개수 제한, 단축키 커스터마이즈.

## 구성

- 신규: `AppSettings.swift` (ObservableObject 싱글턴), `SettingsView.swift`, `SettingsWindowController.swift`
- 수정: `AppDelegate.swift`(메뉴+알림), `StackPanelController.swift`(화면/좌우), `HotkeyManager.swift`(OSStatus)
- 의존성 추가 없음 (ServiceManagement는 시스템 프레임워크). 자동 테스트 없음(기존 스펙 결정 유지), 게이트 = `swift build` + 수동 검증.
